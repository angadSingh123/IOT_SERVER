Query Reference

Query format: COMMAND ARGUMENT PARAMETRES
 
Commands include send, post, ping, echo, close.
Arguments depend on usage of command.
Parametres depend on usage of argument.
---
POST :
 Append to the text box of the receiver.
SEND :
 Send a message. Arguments are attached and so are options.
PING :
 Test the connection.
ECHO: 
 Get a message echoed back from the specified client.
CLOSE:
 Close current connection with the server. Not available on server side.
----
Arguments for send are [Client Number]
Arguments for ping are [Client Number]
Arguments for post are [Client Number]
Arguments for close are none.
Arguments for echo are [Client Number]
---
Parametres for commands with [Client Number] argument: [Message]
---
 
Ex. POST 34 "Hello"
Ex. SEND 34 "ON"
Ex. PING 34
Ex. ECHO 34 "hello"
Ex. CLOSE
Ex. POST SELF "serialwrite:bytes:101001"
Ex. POST SELF "serialwriteline:string:hello"
Ex. POST 65 "serialwrite:bytes:1"
